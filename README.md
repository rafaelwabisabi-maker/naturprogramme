# Naturprogramme

B2B nature pedagogy platform connecting schools and organizations with outdoor education programs in Upper Austria.

## About

Naturprogramme provides structured nature-based learning experiences for schools, municipalities, and youth organizations. Programs are designed around experiential pedagogy principles, delivered on-site in natural environments across Upper Austria. The platform manages program offerings, lead generation, and institutional outreach.

## Tech Stack

- HTML / CSS / JavaScript
- Landing pages
- PDF resources
- GDPR-compliant lead management

## Status

136+ leads collected. Active outreach phase.
